Qu'est-ce que c'est?

Frozen Bubble est un jeu de "match 3" : L'objectif est de regrouper 3 bulles ou plus de même couleur afin de les éliminer. C'est le même concept de base que pour Dr. Mario (match 4), Candy Crush ou beaucoup plus proche, Puzzle Bubble.
Frozen Bubble est un jeu développé en 2003 par Guillaume Cottenceau et qui a depuis été porté sur de nombreuses plateformes. Toutes les informations sur le jeu sont disponibles sur le site officiel.
Ce site est dédié à la version JavaScript de Frozen Bubble, qui reprend les différents modes proposés par Frozen Bubble 2.2.0 directement dans un navigateur. Cette version de jeu est encore en version alpha, tous les modes sont fonctionnels mais de nombreuses anomalies subsistent.

Comment jouer?

Pour le moment, seuls les contrôles clavier sont actifs.
S
W C
[W/Z] = Tourner à gauche, [C] = Tourner à droite, [S] = Tirer

Les contrôles tactiles et la gestion du gamepad sont en cours de développement.

3 modes de jeu

Jeu solo

100 niveaux de difficulté progressive pour apprendre à maîtriser le jeu

Jeu local

A deux sur le même clavier ou seul contre l'ordinateur

Jeu réseau

Le mode ultime, jusqu'à 5 joueurs en réseau
